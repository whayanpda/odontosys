# OdontoSys - Sistema de Agendamento Odontológico 🦷

O **OdontoSys** é um sistema fullstack desenvolvido para o gerenciamento e agendamento de consultas em uma clínica odontológica. O projeto foi idealizado como um MVP (Produto Mínimo Viável) para consolidar conhecimentos em desenvolvimento de software, criar um portfólio robusto e preparar para o mercado de trabalho.

---

## 🎯 Objetivo do Projeto
Desenvolver uma aplicação completa, functional e utilizável, demonstrando domínio na integração entre uma API REST em Java e um frontend dinâmico em React, além de persistência de dados em um banco relacional.

## 🛠️ Tecnologias Utilizadas (Previsão do Projeto)
* **Backend:** Java, Spring Boot, Maven
* **Frontend:** React 
* **Banco de Dados:** PostgreSQL
* **Ferramentas:** Postman, IntelliJ IDEA
